# Sound-translater

## Этот скрипт Python позволяет записывать голос в течение определенного периода времени, сохранять запись в виде файла WAV, распознавать речь из записи с помощью службы распознавания речи Google, а затем переводить распознанный текст на указанный пользователем язык с помощью Google Translate.

## Использование
  1. Запустите сценарий.
  2. При подсказке говорите примерно 5 секунд.
  3. После записи скрипт сохранит аудио для `output.wav` и обработки его.
  4. Вам будет предложено ввести языковой код для перевода (например, `'en'` для английского, `'es'` для испанского).
  5. Сценарий выполняет распознавание речи, а затем переводит признанный текст на выбранный вами язык.
  6. Признанная речь и ее перевод будут отображаться в консоли.
